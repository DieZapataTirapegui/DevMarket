# DevMarket

DevMarket is a production-ready fullstack e-commerce platform built with React (TypeScript) and NestJS.  
It implements JWT authentication, role-based access control, PostgreSQL and Dockerized infrastructure following clean architecture principles.

---

## 🚀 Tech Stack

### Frontend
- React + TypeScript
- Vite
- Tailwind CSS

### Backend
- NestJS
- PostgreSQL
- TypeORM
- JWT Authentication
- Docker

---

## 📦 Features (Planned)

- User registration & login
- JWT authentication
- Role-based access control (Admin / User)
- Product management (CRUD)
- Shopping cart
- Order system
- Admin dashboard
- Dockerized database
- API documentation with Swagger
